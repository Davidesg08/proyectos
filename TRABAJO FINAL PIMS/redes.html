02

Tutorial DHCP — Configuración Completa

El Protocolo de Configuración Dinámica de Host (DHCP) automatiza la asignación de parámetros de red. A continuación se documenta la configuración en Windows Server 2022 y en Cisco IOS.

📋 Pasos de Configuración — Windows Server DHCP

Rol DHCP Server en Windows Server 2019/2022

01
Instalar el rol DHCP

Desde Server Manager → Add Roles and Features → DHCP Server. Completar el asistente de configuración post-instalación.

02
Crear un Scope (Ámbito)

DHCP Manager → Expand server → IPv4 → New Scope. Definir nombre, rango de IPs, máscara, exclusiones y duración del lease.

03
Configurar opciones del scope

Scope Options → Definir: 003 Router (gateway), 006 DNS Servers, 015 DNS Domain Name, 044 WINS Servers (si aplica).

04
Crear Reservas (IPs Fijas)

Reservations → New Reservation → Ingresar nombre, IP deseada y MAC address del cliente. Útil para impresoras y servidores con DHCP.

05
Activar el scope y autorizar el servidor

Clic derecho sobre el scope → Activate. Luego autorizar el servidor en el dominio AD: Clic derecho en el servidor → Authorize.

Scripts PowerShell — Red

powershell — inventario de red
# ============================================ # Script: Get-NetworkInventory.ps1 # Autor: Equipo IT NextGen # Versión: 1.3 # ============================================ param( [string]$Subnet = "192.168.20", [int]$StartHost = 1, [int]$EndHost = 254, [string]$OutputPath = "C:\IT\Reports\NetworkScan.csv" ) $Results = @() foreach ($i in $StartHost..$EndHost) { $ip = "$Subnet.$i" $ping = Test-Connection -ComputerName $ip -Count 1 -Quiet if ($ping) { try { $hostname = (Resolve-DnsName $ip -ErrorAction Stop).NameHost } catch { $hostname = "N/A" } $Results += [PSCustomObject]@{ IP = $ip Hostname = $hostname Status = "Online" Timestamp = Get-Date -Format "yyyy-MM-dd HH:mm" } } } $Results | Export-Csv -Path $OutputPath -NoTypeInformation Write-Host "Escaneo completo. Hosts activos: $($Results.Count)" -ForegroundColor Green
